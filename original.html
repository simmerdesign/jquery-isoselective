Description:

isoSelective is a plugin written specifically to extend the functionality of jQuery Isotope. I'm sure when David / Metafizzy created Isotope, he wanted to keep the codebase as light as possible, while still providing a huge chunk of robust functionality.

By default,Isotope has a powerful and intelligent filtering system. isoSelective attempts to expand on the wonderful foundation, by providing a small jQuery plugin for more advanced filtering.

What does isoSelective do?:

Simply put: isoSelective transforms your filtering links into a dynamic system allowing for on-the-fly creation of new filtering strings, which allow any item to be toggled on / off.

Imagine the following: You have a category of pictures you want users to be able to filter by category. You have the categories ‘Black and White’, ‘Last Month’, and lastly ‘Portraits’.

Using the default Isotope filtering system, it’s easy to setup links for each category that will show it, and hide the others. But what if you wanted users to be able to view BOTH ‘Last Month’ and ‘Portraits’ for example. You’d have to add another filter containing the filters of both the ‘Last Month’ AND the ‘Portraits’ links.

Adding one link is no big deal. But let’s expand. Imagine you had 10 categories of photos, and wanted any category to be shown with any others the user wants. This would require, using the default Isotope system, over 1000 filter links.

Using isoSelective bypasses this tedious restriction, and creates a setup where each filter can be clicked to turn it on, and clicked again to turn off. In the example, the original 10 links would be all that is needed. If a user wanted to view two categories, they would simply toggle the two they wished to see on, and toggle the others off. Alternatively, they could toggle an ‘All’ filter link, and then toggle on the two categories.

Usage:

Using isoSelective is easy.Following your call to create the Isotope layout, you add a call to isoSelective. This call requires 3 parameters.


 $container.isoSelective({
   linkSelector: '.filter',
   attrSelector: 'rel',
   activeClass: 'selected'
 });


linkSelector: This is the selector for your filter links. Default: '.filter' - Example: linkSelector: '.filter' would tell isoSelective that every element with a class of 'filter' is used for filtering.

attrSelector: This is used to specify the selection attribute. You can use pretty much any nonsense attribute you want. Default: 'rel' - Example: attrSelector: 'rel' would tell isoSelective to look for the value in the 'rel' attribute.

activeClass: This is the class you used and want applied to elements that are toggled 'On.' If you are not filtering anything before you make your call to isoSelective, your original filter links should all be in the on state, and have this class. If you are filtering before calling isoSelective to process links, then you should give the items shown's corresponding filter links this value in your xhtml. Default: 'selected' - Example: activeClass: 'selected' tells isoSelective that any filter links with the class 'selected' are on, those without are off, and to add the class or remove it when the element is turned on or off.

Sample Code:

<ul id="filters" class="clearfix">
               <li>
                    <a href="#filter" rel="*" class="selected">all</a>
               </li>
               <li>
                    <a href="#filter" rel=".blue">blue</a>
               </li>
               <li>
                    <a href="#filter" rel=".green">green</a>
               </li>
               <li>
                    <a href="#filter" rel=".red">red</a>
               </li>
   </ul>
   <div id="container" class="clearfix">
               <div class="element red blue">
                    <p>
                         Red and Blue Element
                    </p>
               </div>
               <div class="element red">
                    <p>
                         Red Element
                    </p>
               </div>
               <div class="element red green">
                    <p>
                         Red and Green Element
                    </p>
               </div>
               <div class="element green">
                    <p>
                         Green Element
                    </p>
               </div>
               <div class="element blue green">
                    <p>
                         Blue and Green Element
                    </p>
               </div>
               <div class="element blue">
                    <p>
                         Blue Element
                    </p>
               </div>
   </div>

 <script src="js/jquery-1.4.4.min.js"></script>
 <script src="js/jquery.isotope.min.js"></script>
 <script src="js/jquery.isoSelective.min.js"></script>
 <script>
  
    var $container = $('#container');




    $(function(){
      $container.isotope({
        itemSelector : '.element'
      });
  
  $container.isoSelective({
        linkSelector: '#filters a',    // this is the filter link 
        attrSelector: 'rel',           // the attribute containing the filter
        activeClass: 'selected'        // class to add to selected
    });
      
  });
  
 </script>
